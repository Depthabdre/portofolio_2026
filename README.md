# Abdrehim Misbah — Portfolio 2026

A modern, highly interactive software engineering portfolio built to showcase my backend and mobile development projects, technical journey, and personal philosophy. Designed with an ultra-minimalist OLED dark theme, frosted glass bento grids, and smooth physics-based animations.

## ✨ Features
- **Immersive Animations**: Physics-based spring animations, cubic-bezier scroll reveals, and a dynamic "tracing beam" timeline powered by Framer Motion.
- **Bento Grid Stats**: A sleek, hover-responsive bento grid tracking my academic and technical milestones.
- **Cinematic Project Showcases**: Features horizontal auto-scrolling UI rails displaying high-res mobile app interfaces.
- **Infinite Tech Marquee**: A customized, CSS-masked continuous marquee strip highlighting my core stack.
- **OLED-Optimized Dark Mode**: Deep midnight backgrounds with glowing cyan (`#54c5f8`) and lime accents to emphasize core developer competencies.

## 🛠️ Tech Stack
- **Framework**: [Next.js](https://nextjs.org/) (App Router)
- **Library**: [React 18](https://react.dev/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Motion & Physics**: [Framer Motion](https://www.framer.com/motion/)
- **Hosting**: [Vercel](https://vercel.com/)

## 🚀 Getting Started

To run this project locally:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Depthabdre/portofolio_2026.git
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. **Open your browser:**
   Navigate to `http://localhost:3000` to view the site.
